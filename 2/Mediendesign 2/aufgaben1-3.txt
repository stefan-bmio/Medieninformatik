Verschiedene Brennweiten
Erstellen Sie eine Bildreihe, in der Sie mittels fünf verschiedener Brennweiten unterschiedliche Aspekte einer Szene visuell beschreiben. Benutzen Sie ein Teleobjektiv, ein leichtes Tele, ein Normalobjektiv, ein leichtes Weitwinkel- und ein Weitwinkelobjektiv. Diese Brennweiten können Sie gerne an einem Zoom-Objektiv ausprobieren.

Verändern Sie den Standpunkt der Kamera nicht, wechseln Sie lediglich die Brennweiten und beobachten Sie die unterschiedliche Wirkung des Bildraumes. Das Motiv soll sich dabei wenig verändern, die unterschiedlichen Bildausschnitte sollten aber jeweils eine unterschiedliche Bildwirkung zeigen.

Präsentieren Sie die Bildreihe als PDF.

Wasser in Bewegung
Fotografieren Sie Wasser in Bewegung. Versuchen Sie zuerst, das Wasser in seiner Bewegung einzufrieren, indem Sie eine sehr kurze Verschlusszeit wählen. Achten Sie darauf, dass Sie genügend Licht zur Verfügung haben, sonst können Sie keine kurzen Zeiten wählen. In einem zweiten Versuch lassen Sie den Verschluss länger offen, damit das Wasser verschwimmt. Benutzen Sie ein Stativ, sonst verwackelt die Aufnahme.

Tiefenschärfe
Zeigen Sie anhand von einer Einstellung (natürlich mit Stativ aufgenommen!), wie unterschiedlich die Bildwirkung sein kann, wenn wir

eine offene Blende wählen, und damit nur einen bestimmten Bildteil scharf erscheinen lassen und
die Blende möglichst weit abblenden, um das Bild von vorne bis hinten scharf erscheinen zu lassen.
Achten Sie bei dieser Aufgabe auf ausreichend Licht, sonst kann 2. nicht gelingen.
Versuchen Sie, vorher die Aufnahme so zu konzipieren, dass Sie voraussehen, welche Bildwirkung eintreten wird.

Thema der Bilder soll sein: »Fremd«
